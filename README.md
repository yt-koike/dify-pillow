## dify-pillow

**Author:** yt-koike
**Version:** 0.0.1
**Type:** tool

### Description

dify-pillow is a standalone plugin for dify to enable image editing with a help of a Python library called [Pillow](https://pillow.readthedocs.io/en/stable/index.html).
